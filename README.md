# Flappy Bird

You all know the game. This is just a rust implementation of it to test out the Bevy game engine.

I followed [this tutorial](https://www.youtube.com/watch?v=jKSZ8k70dxM). Very good and you end up with a pretty polished project in a couple hundred lines of code. I think this is a great beginner game. The tutorial glossed over some stuff I felt, but mostly things like the math and shaders that aren't super important or exclusive to Bevy itself. I'm totally new to game dev stuff, so they're probably pretty simple things that I just don't know.

Their source code can be found [here](https://github.com/rust-adventure/flappy-bird/)
